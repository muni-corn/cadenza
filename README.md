# cadenza

hello! welcome to the **cadenza** repository!

**cadenza** is a collection of projects that make the Linux desktop more beautiful and delightful to use.

it's ***shamelessly the most beautiful*** collection of Linux utilities and enhancements. or, at least, that's been its tagline for as long as i can remember.

many things here are still under heavy development, so **please open up issues** if you want any features or bug fixes prioritized! 

## about the projects

### cadenza shell

the firstborn of the cadenza project, previously under the names of `muse-status` and `muse-shell`.

### cadenza sounds

a pleasing sound theme for your desktop that will spark the dopamine you need to be productive.

### pinentry-cadenza

why hasn't anyone come up with a better-looking **pinentry** program that works on all Wayland compositors? 

`pinentry-cadenza` will charm you with its aesthetics and its functionality. a simple password prompt has never been so delightful to use as this.

pairs *very* well with **cadenza sounds** for special authentication dialog chimes. yeah, that's right. it makes sound.~ that's what you can expect from the music unicorn.
